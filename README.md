🌞 Embry-Riddle Solar Telescope Forecast Dashboard (STFD)

The Embry-Riddle Aeronautical University Solar Telescope Forecast Dashboard (STFD) is a Python-based web application designed to assist ERAU solar forecasters before, during, and after observation sessions.

Built with SunPy (https://sunpy.org/) and Streamlit (https://streamlit.io/), the dashboard provides a simple, interactive interface that brings together key tools for solar data visualization, forecasting, and observation planning all in one place.

🔭 Project Overview

The STFD was developed to help solar observers quickly determine:
- When to observe: Based on current and predicted solar activity.
- Where to focus: Using real-time imagery and magnetic field data.
- What to look for: Including coronal holes, filaments, active regions, and flares.
By combining these features into an accessible dashboard, STFD aims to improve efficiency and decision-making during limited solar telescope time, ensuring that each observation period is used to its fullest potential.

🧰 Tech Stack: Python
- SunPy: Solar data retrieval and analysis
- Streamlit: Interactive web-based dashboard
- Matplotlib / NumPy / Pandas: Data handling and visualization

🚀 Goals
- Provide real-time solar forecasting tools for ERAU observers
- Streamline pre-observation planning and in-session monitoring
- Encourage open-source collaboration for solar data visualization and forecasting
